프로젝트 설명
11-13주차 실습에서 구현한 미로를 바탕으로, 미로의 각 모서리로부터 미로의 중앙까지 도달하는 데 필요한 거리를 구간별로 나누어 Graphic 요소를 가미해 보인다.
5초마다 새 미로를 생성해 새로운 결과를 출력한다.

프로젝트 실행 환경
 - Visual Studio 2017
 - openFrameworks version 0.11.2
 - 출력 모니터 디스플레이 해상도 2560*1600
   (프로젝트를 실행하는 모니터 환경에 따라 화면에 그려지는 결과가 상이할 수 있으나, ofApp.h 파일에 정의된 매크로의 값을 실행 환경에 맞게 수정해 정상 출력 가능 / XS - 선 길이 배율, resolution_x - 실행할 window의 x축 길이, resolution_y - 실행할 window의 y축 길이)
